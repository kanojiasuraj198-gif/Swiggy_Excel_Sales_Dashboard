# Swiggy_Excel_Sales_Dashboard_Project
# Project Overview
This project focuses on analyzing food delivery sales data to track business performance, customer behavior, and regional trends. The dashboard provides actionable insights using key performance indicators (KPIs) and interactive visualizations.
# Business Objectives
- Monitor overall sales performance
- Understand customer satisfaction through ratings
- Identify high-performing regions, cities, and food categories
- Analyze sales trends across time (daily, weekly, monthly, quarterly)
- Support data-driven business decisions
